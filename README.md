# Superstore Sales Analysis Dashboard 📊

## Overview
This project presents an interactive **Superstore Sales Analysis Dashboard** developed to analyze sales performance, customer segments, product performance, and geographical trends. The dashboard provides key business insights that help understand revenue patterns, identify top-performing products and regions, and support data-driven decision-making.

## Dashboard Highlights
- **Total Sales KPI:** Displays overall revenue generated from sales.
- **Total Orders KPI:** Tracks the number of orders processed.
- **Average Shipping Days:** Measures delivery efficiency.
- **Monthly Sales Analysis:** Identifies seasonal trends and monthly revenue fluctuations.
- **Customer Segment Analysis:** Compares sales contribution from Consumer, Corporate, and Home Office segments.
- **Category Performance:** Analyzes sales distribution across Technology, Furniture, and Office Supplies.
- **Top 10 Products by Sales:** Highlights the highest revenue-generating products.
- **Top 10 Cities by Sales:** Shows cities contributing the most to overall sales.
- **Interactive Filters:** Allows users to analyze data by country/region, order date, and shipping date.

## Tools & Technologies Used
- Microsoft Excel / Power BI (Dashboard Development)
- Data Cleaning & Transformation
- Pivot Tables & Data Analysis
- Data Visualization Techniques

## Business Insights
- Identifies major revenue drivers and high-performing categories.
- Helps understand customer purchasing behavior.
- Highlights top products and locations for strategic focus.
- Provides insights into sales trends and operational efficiency.

## Objective
The main objective of this dashboard is to transform raw sales data into meaningful visual insights that help businesses monitor performance, discover growth opportunities, and improve decision-making.

## Dataset
The analysis is based on the Superstore sales dataset containing information about orders, customers, products, sales, shipping details, and geographic locations.
